1 : Create a ext4  partition


2.Take a backup of home folder
->sudo cp -Rp /home/* /mount/location


3.Locate UUID of the Disk

->  sudo blkid	 ( note the output)

4.Open fstab and add the following entry

UUID= (one we noted of the new device)     /home     ext4     nodev,nosuid     0     2


5.Remove old home and add new home folder

cd / && sudo mv /home /home_old && sudo mkdir /home  



6.Mount /home to new partition 

mount /dev/sdb1  /home  # ssb1 should be chnaged to the one you have created.


7.Reboot the server( It may take few min to reboot)
